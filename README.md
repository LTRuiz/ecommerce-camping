🏔️ Patagonia Camp
Patagonia Camp es un boceto de e-commerce de artículos de camping desarrollado con tecnologías web base. El objetivo principal es mostrar una estructura de archivos organizada y una lógica de carrito funcional mediante la manipulación del DOM.

🛠️ Tecnologías
HTML5

CSS3

JavaScript (Vanilla)

📂 Estructura
El proyecto mantiene una organización estándar de desarrollo web:

index.html: Estructura principal.

css/: Estilos del sitio.

img/: Imágenes de productos.

js/: Lógica y funciones.

🚀 Instalación
Clona el repo: git clone https://github.com/LTRuiz/patagonia-camp.git

Abre index.html en tu navegador.
